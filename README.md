# Enterprise Sales Analytics & BI Reporting

## Overview
This project focuses on analyzing enterprise sales data using Power BI to track sales performance, returns, and regional trends.

## Tools & Technologies
- Power BI
- Power Query
- DAX
- SQL

## Key Business Questions
- How is sales performance trending across regions?
- Which categories and segments drive profitability?
- How do actual sales compare against targets?

## Key Features
- Interactive dashboards
- Target vs Actual KPI analysis
- Regional and category-level insights

## Dataset
Global Superstore dataset

## Deliverables
- Power BI dashboard
- KPI visuals and insights
