## Data Files

This folder contains supporting files used for building the Power BI dashboard in this project.

### Files Included:
- **Global_SuperStore.pbix**  
  Power BI report file containing data model, transformations, DAX measures, and dashboard visuals.

- **Report_Analysis_Template.xlsx**  
  Analysis planning template used for understanding requirements, defining KPIs, and structuring the dashboard before development.

These files support the end-to-end BI development process demonstrated in this repository.
