## Dashboard Screenshots

This folder contains screenshots of the Power BI dashboard pages created as part of the Enterprise Sales Analytics project.

### Screenshot Order:
1. **Data Model** – Shows relationships between fact and dimension tables.
2. **Executive Overview** – High-level business performance summary.
3. **Sales Profit Trend** – Time-based analysis of sales and profit.
4. **Regional Sales Analysis** – Region and state-wise sales performance.
5. **KPI Performance Summary** – Key performance indicators and target tracking.

These screenshots are referenced in the main README to explain dashboard insights and business analysis.
